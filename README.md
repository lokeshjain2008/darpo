# Darpo - Hotel Management System

A modern hotel management system built with Next.js, Supabase, and Expo.

## Tech Stack

- Next.js
- Supabase
- Expo
- TypeScript

## Project Structure

```
.
├── apps
│   ├── web     # Next.js application
│   └── mobile  # Expo application
└── packages
    └── shared  # Shared utilities and types
```
